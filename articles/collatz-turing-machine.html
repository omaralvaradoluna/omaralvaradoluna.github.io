> ANÁLISIS: ¿CUÁNTA MEMORIA NECESITA EL CAZADOR?
PREGUNTA: ¿Puede el cazador operar con solo 1 bit de memoria?
RESPUESTA: NO. Necesita al menos 3 bits (5-6 estados).

Razón: El cazador debe distinguir entre fases y trackear carry:
FASE SEEK: Buscar LSB, decidir par/impar FASE DIV: Borrar ceros (÷2) FASE MUL: Aplicar 3n+1 (requiere carry) FASE BACK: Regresar al LSB
> AUTÓMATA MÍNIMO DEL CAZADOR (5 ESTADOS)
> TABLA DE REGLAS DEL CAZADOR (VISIÓN LOCAL)
ESTADO PREV CURR NEXT →ESCRIBE →MUEVE →ESTADO NOTA
SEEK * 0 * - → DIV PAR: dividir
SEEK * 1 * - → MUL:C=1 IMPAR: 3n+1
SEEK * □ * - - HALT CINTA="1" ✓
DIV * 0 * □ → DIV BORRAR CERO
DIV * 1 * - ← SEEK NUEVO LSB
MUL:C=0 0 0 * 0 → MUL:C=0 2·0+0+0=0
MUL:C=0 1 0 * 1 → MUL:C=0 2·0+1+0=1
MUL:C=0 0 1 * 0 → MUL:C=1 2·1+0+0=2
MUL:C=0 1 1 * 1 → MUL:C=1 2·1+1+0=3
MUL:C=1 0 0 * 1 → MUL:C=0 2·0+0+1=1
MUL:C=1 1 0 * 0 → MUL:C=1 2·0+1+1=2
MUL:C=1 0 1 * 1 → MUL:C=1 2·1+0+1=3
MUL:C=1 1 1 * 0 → MUL:C=2 2·1+1+1=4 !
MUL:* * □ * [C] ← BACK FIN: emitir carry
BACK * 0,1 * - ← BACK REBOBINAR
BACK * □ * - → SEEK NUEVO CICLO
OBSERVACIÓN CLAVE:
Las reglas de multiplicación solo dependen de: (PREV, CURR, CARRY)
El bit NEXT no se usa para decidir qué escribir — solo se mira al avanzar.

FÓRMULA UNIVERSAL:
WRITE = (2·CURR + PREV + CARRY) mod 2
CARRY' = ⌊(2·CURR + PREV + CARRY) / 2⌋
> LA ESENCIA DEL CAZADOR
TEOREMA: MEMORIA MÍNIMA DEL CAZADOR
El cazador Collatz NO puede operar con 1 solo bit de memoria.
Requiere al menos 3 bits (5 estados efectivos):

{ SEEK, DIV, MUL:C=0, MUL:C=1, BACK }

Sin embargo, el carry puede llegar a 2 en casos extremos (1+1+1=3→1,C=2)
lo que agrega un 6° estado: MUL:C=2
INSIGHT PROFUNDO: El cazador es un autómata finito determinista que procesa una cinta infinita.
Pero su complejidad no está en él — está en la estructura de los números que recorre.
La pregunta Collatz es: ¿toda cinta eventualmente colapsa a "1"?

> CONDICIÓN DE HALT: "¡A LA DERECHA NO HAY MÁS!"
HALT = (STATE == SEEK) ∧ (PREV == □) ∧ (CURR == 1) ∧ (NEXT == □)
INTERPRETACIÓN:
"Estoy buscando qué hacer (SEEK), veo un 1 solitario rodeado de vacío"
→ ¡La cinta contiene exactamente el número 1! → TERMINAMOS
